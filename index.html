Real-time Phosphorus Risk Prediction Dashboard

Early Warning System for Phosphorus Spikes in WWTP using SCADA Data & Deep Learning

2025
Community Project 4 months

Project Overview

Developed an AI-powered early warning system to predict phosphorus spikes in wastewater treatment plants using high-frequency SCADA data. The system employs Bidirectional LSTM (BiLSTM) deep learning models to provide real-time risk predictions, helping prevent eutrophication in receiving water bodies.

Problem Context

  • Phosphorus discharge from WWTPs causes eutrophication in rivers and lakes
  • Short-term phosphorus spikes often go undetected by conventional monitoring
  • Manual monitoring at plants like Mokshdham STP (Nagpur) leads to reactive responses
  • Regulatory compliance based on end-of-pipe measurements misses transient events

Data Sources & Methodology

Agtrup WWTP Dataset (Denmark)
Mokshdham STP Field Study (Nagpur)
Bidirectional LSTM
Time-Series Analysis

Technical Implementation

Data Preprocessing: Handled 40,000+ samples at 2-minute resolution using LOCF for missing values
Feature Engineering: Process-informed selection of 23 variables including influent flow, DO, temperature, and phosphate rates
Temporal Windowing: Sliding windows to capture process dynamics and delayed responses
Risk-Based Formulation: Binary classification for spike prediction vs. continuous concentration forecasting

Model Performance

Event Detection Performance
Recall: 84% Spike Detection Rate
Discrimination Ability
ROC-AUC: 0.85 Better than Random

Field Study: Mokshdham STP, Nagpur

Site Observations

Conducted field visit to 5 MLD Mokshdham Sewage Treatment Plant operated by Nagpur Municipal Corporation. Key findings:

  • Total flow: 4220 m³/day, COD removal: 92%, BOD removal: 30%
